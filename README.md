# Selenium Automation Portfolio

This repository contains personal automation projects built using Python and Selenium WebDriver.

## 🚀 Projects

### 1. Google Search Automation
Automates user behavior on Google Search:
- Open Google
- Enter search keyword
- Submit search
- Click target result link

### 2. Cookie Clicker Bot
Automation bot for Cookie Clicker:
- Automatically clicks cookie
- Reads current cookie count
- Purchases upgrades automatically
- Uses dynamic waits for web elements

## 🛠️ Tech Stack
- Python
- Selenium WebDriver
- Chrome Browser
- Explicit Wait (WebDriverWait)
